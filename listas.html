Funcion
Una función es un bloque de código que realiza alguna operación.
let
La instrucción let declara una variable de alcance local con ámbito de bloque, la cual, opcionalmente, puede ser inicializada con algún valor.
const
Las variables constantes presentan un ámbito de bloque tal y como lo hacen las variables definidas usando la instrucción let, con la particularidad de que el valor de una constante no puede cambiarse a través de la reasignación. Las constantes no se pueden redeclarar.
